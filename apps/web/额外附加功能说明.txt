额外附加功能说明

Shiroha Quiz Web 版默认使用 CDN 加载以下扩展资源。如果希望离线使用或加载更快，请运行对应的下载脚本（在 libs/ 目录中）。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

一、PDF.js —— PDF 文字层解析

加载顺序：本地 libs/pdf.min.mjs → CDN jsDelivr → CDN unpkg → 内置轻量提取器

完整离线包包含：
  - 核心库（pdf.min.mjs / pdf.worker.min.mjs）
  - CMaps（中日韩字符映射）
  - 标准字体（Foxit / Liberation）
  - ICC 色彩配置
  - WASM 图片解码（JBIG2 / OpenJPEG / QCMS）
  - 图片解码器

下载：libs/download-pdfjs.ps1

注意：完整包增强的是文字层 PDF 识别稳定性与图片解码质量，不等于 OCR。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

二、MathJax —— 公式渲染

加载顺序：本地 libs/mathjax/ → CDN jsDelivr → 降级文本显示

完整离线包包含 MathJax 3.2.2 完整 es5 组件。

下载：libs/download-mathjax.ps1（约 24 MB）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

三、OCR —— 扫描件兜底

用途：
  - 图片型 PDF / 扫描 PDF 的兜底识别
  - OCR 结果可预览、填入导入区、复制或下载为 DOCX
  - 填入后继续走 Shiroha Quiz 原有题库解析流程

触发条件：仅在文字层 PDF 提取失败或文字过少时显示 OCR 面板。

完整离线包包含：
  - Tesseract.js 5.1.1 入口（tesseract.min.js / worker.min.js）
  - tesseract.js-core WASM 引擎
  - 简体中文语言包（chi_sim）
  - 英文语言包（eng）

下载：libs/ocr/download-ocr.ps1（约 59 MB）

注意事项：
  - OCR 是兜底方案，不会自动替代文字层 PDF 解析。
  - OCR 结果可能有错字、漏字和段落错位，导入题库前必须人工核对。
  - OCR 生成 DOCX 是可编辑文本版，不承诺还原原始排版。
  - 部分浏览器会限制 file:// 页面加载本地 Worker，如离线 OCR 失败，建议通过本地 HTTP 服务打开 Web 版（如 npx serve .）。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

通用说明：
  - 所有扩展功能不运行脚本也可正常使用，CDN 会自动兜底。
  - 离线包下载后放在 libs/ 目录下即生效，无需额外配置。
  - 如需恢复纯 CDN 模式，删除对应目录即可。
