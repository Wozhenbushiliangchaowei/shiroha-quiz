# Contributing

感谢你关注 Shiroha Quiz。

当前项目仍处于早期阶段，欢迎通过 Issue 或 Pull Request 参与改进。

## 你可以帮助的方向

- Bug 反馈
- 功能建议
- 题库导入兼容性问题
- UI / 交互优化建议
- 文档改进建议

## 提交建议

在提交 Pull Request 前，建议先通过 Issue 说明以下内容：

- 你准备解决什么问题
- 你的修改范围是什么
- 是否会影响 Web、Android 或共享逻辑

## 当前优先方向

当前更欢迎这些方向的改进：

- Web 题库导入逻辑稳定性
- Android 原生刷题体验
- 文档和目录结构完善
- 示例题库和使用说明补充

## 代码与文档约定

- 尽量保持目录边界清晰
- 不要把平台专属逻辑随意混入共享层
- 文档默认优先补中文
- 涉及公开上传时，避免引入任何敏感、专有或个人信息
