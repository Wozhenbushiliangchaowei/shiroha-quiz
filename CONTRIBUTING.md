# Contributing

感谢你关注 `Shiroha Quiz`。

当前版本：统一发布 `v2.6.0-beta`，原生 Compose `v0.8.2-native`，Web `v0.7.1-alpha`（V35）。原生 Compose 版为主要开发线，Web 端同步演进。

欢迎通过 Issue 或 Pull Request 参与改进。

## 可以帮助的方向

- Bug 反馈
- 题库导入与解析兼容性问题
- Web 端 UI / 交互优化
- Android 原生体验优化
- 文档与示例题库补充
- 跨端数据互通测试

## 提交前建议

在提交 Pull Request 前，建议先通过 Issue 说明：

- 你准备解决什么问题
- 改动主要影响 `Web`、`Android` 还是共享逻辑
- 是否会影响题库解析、导入结果或已有数据结构
- 回归测试 21 用例是否仍通过

## 当前优先方向

目前更欢迎这些方向的改进：

- 原生解析器样例与真实题库回归
- 导入格式文档完善
- 跨端数据互通与备份兼容性
- Compose UI 体验优化
- 大题库（500+）导入性能优化
- CI 集成回归测试自动化

## 代码与文档约定

- 默认统一项目名称为 `Shiroha Quiz`
- 尽量保持目录边界清晰
- 不要把平台专属逻辑随意混入共享层
- 涉及导入解析逻辑时，优先补样例和验证说明
- 涉及公开上传时，避免引入任何敏感、专有或个人信息
- Native 端 `test/native-parser-regression/` 为解析回归测试，解析变更需验证 21/21 通过
