# Shiroha Quiz

`Shiroha Quiz` 是一个面向多题库场景的通用刷题项目。

当前目标不是只做一个网页工具，而是逐步发展成一套可扩展的多端刷题系统：

- `Web` 端用于快速导入、管理、验证题库能力
- `Android` 端用于原生、高质感、重体验的刷题与考试
- 后续核心逻辑会继续沉淀到共享模块中

## 当前仓库结构

```txt
shiroha_quiz/
├─ apps/
│  ├─ web/        # 当前通用网页版参考实现
│  └─ android/    # 安卓原生工程（Jetpack Compose）
├─ packages/
│  ├─ core/       # 后续沉淀刷题核心逻辑
│  ├─ parser/     # 后续沉淀题库导入与解析逻辑
│  ├─ types/      # 后续沉淀通用数据结构
│  ├─ shared/     # 后续沉淀通用工具能力
│  └─ ui/         # 后续沉淀跨端或共享 UI 规范
├─ docs/          # 产品、设计、架构文档
├─ assets/        # 图标、展示图、设计素材
└─ archive/       # 不参与版本管理的本地归档区
```

## 当前重点

当前主线分两条：

1. `apps/web`
继续作为题库导入能力和数据格式验证参考

2. `apps/android`
继续推进原生安卓体验，优先完成：
- 首页
- 练习页
- 导入页
- 状态流与题库模型

## 导入能力目标

安卓首版目标支持：

- `txt`
- `json`
- `csv`
- `docx`
- `doc`
- 文字层 `pdf`
- 双文件导入

当前阶段暂不做 OCR PDF。
