# 标准题库格式示例

当题库导入失败、题目识别不完整、答案无法匹配时，建议先将题库整理成以下标准格式，再重新导入。

核心原则：**一题一段、题号清晰、选项换行、答案单独一行、解析可选。**

## 一、通用格式要求

每道题建议使用以下结构：

```text
题号. 题干
A. 选项一
B. 选项二
C. 选项三
D. 选项四
答案：A
解析：这里填写解析内容，没有解析可以不写
```

整理时请尽量遵守：

1. 每道题之间空一行。
2. 题号使用 `1.`、`2.`、`3.` 这种格式。
3. 选项单独换行，使用 `A.`、`B.`、`C.`、`D.`。
4. 答案单独一行，统一写成：`答案：A`。
5. 多选题答案写成：`答案：ABC` 或 `答案：A,B,C`。
6. 判断题答案写成：`答案：正确` 或 `答案：错误`。
7. 填空题、简答题也要单独写答案。
8. 解析不是必须项，但有解析时统一写成：`解析：……`。

## 二、单选题标准格式

```text
1. 下列哪一项属于安全操作要求？
A. 按规定佩戴防护用品
B. 随意拆除安全装置
C. 在设备运行时进行违规操作
D. 未经确认直接启动设备
答案：A
解析：安全操作应当遵守现场规定，并正确佩戴防护用品。
```

## 三、多选题标准格式

```text
2. 下列哪些属于作业前检查内容？
A. 检查设备状态
B. 检查工具是否齐全
C. 检查现场安全条件
D. 忽略风险提示直接作业
答案：ABC
解析：作业前应检查设备、工具和现场安全条件。
```

多选题答案可以写成：

```text
答案：ABC
```

也可以写成：

```text
答案：A,B,C
```

但建议全文保持一种格式。

## 四、判断题标准格式

```text
3. 作业前应进行安全确认，确认无误后方可开始作业。
答案：正确
解析：作业前安全确认是基本要求。
```

也可以使用：

```text
答案：错误
```

不建议混用：

```text
对 / 错
√ / ×
✔️ / ✖️
T / F
```

虽然系统可能能识别其中一部分格式，但为了稳定导入，建议统一为：

```text
正确 / 错误
```

## 五、填空题标准格式

```text
4. 作业人员进入现场前，应按要求佩戴____。
答案：防护用品
解析：进入现场前应按规定佩戴劳动防护用品。
```

填空题建议题干中保留明显空格标记，例如：

```text
____
（ ）
( )
```

## 六、简答题标准格式

```text
5. 简述作业前安全检查的主要内容。
答案：作业前应检查设备状态、工具准备情况、现场环境、安全防护措施以及人员分工情况，确认满足作业条件后方可开始作业。
解析：简答题答案可以是一句话，也可以是一小段文字。
```

简答题答案较长时，不要拆成多个题目。建议完整放在同一个“答案：”后面。

## 七、带分区的标准格式

如果题库包含多个题型，建议使用清晰分区。

```text
一、单选题

1. 下列哪一项属于正确操作？
A. 按流程作业
B. 随意操作
C. 忽略检查
D. 擅自修改参数
答案：A

2. 作业前首先应进行什么？
A. 安全确认
B. 直接启动设备
C. 跳过检查
D. 离开现场
答案：A

二、多选题

1. 下列哪些属于安全检查内容？
A. 设备状态
B. 工具情况
C. 现场环境
D. 无关操作
答案：ABC

三、判断题

1. 作业前可以不进行风险识别。
答案：错误

2. 发现异常情况应及时报告。
答案：正确
```

注意：分区题库中，每个分区可以重新从 `1.` 开始编号，但分区标题必须清楚。

推荐分区标题：

```text
一、单选题
二、多选题
三、判断题
四、填空题
五、简答题
```

## 八、不推荐的格式

### 1. 题干和选项挤在一行

不推荐：

```text
1. 下列哪项正确？A. 选项一 B. 选项二 C. 选项三 D. 选项四 答案：A
```

建议改成：

```text
1. 下列哪项正确？
A. 选项一
B. 选项二
C. 选项三
D. 选项四
答案：A
```

### 2. 答案集中在最后但题号混乱

不推荐：

```text
1. 题目……
2. 题目……
答案：1A 2B 3C
```

建议整理为清晰答案区：

```text
参考答案：
1. A
2. B
3. C
```

更推荐直接写进每道题：

```text
1. 题目……
答案：A
```

### 3. 选项没有 A/B/C/D 标记

不推荐：

```text
1. 下列哪项正确？
选项一
选项二
选项三
选项四
答案：选项一
```

建议改成：

```text
1. 下列哪项正确？
A. 选项一
B. 选项二
C. 选项三
D. 选项四
答案：A
```

## 九、可以交给 AI 工具清洗的提示词

可以把原始题库复制给 AI 工具，然后使用下面的提示词：

```text
请将下面的题库整理成标准导入格式。

要求：
1. 每道题单独成段。
2. 题号统一使用 1. 2. 3. 格式。
3. 单选题和多选题的选项统一使用 A. B. C. D. 格式，并且每个选项单独换行。
4. 答案统一写成“答案：A”这种格式。
5. 多选题答案统一写成“答案：ABC”。
6. 判断题答案统一写成“答案：正确”或“答案：错误”。
7. 填空题保留题干中的空格符号，例如 ____。
8. 简答题答案完整保留，不要随意拆分。
9. 有解析就写“解析：……”，没有解析可以省略。
10. 不要改写题干和选项含义，只做格式清洗。
11. 不要添加原文中不存在的新题目。
12. 输出纯文本，不要输出表格。

下面是原始题库：
【把题库粘贴到这里】
```

## 十、最推荐的最终格式

最稳定的导入格式就是：

```text
1. 题干内容
A. 选项一
B. 选项二
C. 选项三
D. 选项四
答案：A
解析：可选内容

2. 题干内容
A. 选项一
B. 选项二
C. 选项三
D. 选项四
答案：BCD
解析：可选内容

3. 判断题题干内容。
答案：正确

4. 填空题题干____。
答案：填空内容

5. 简答题题干。
答案：简答题答案内容
```

一句话总结：**题号清楚、选项换行、答案单独写、题型分区清楚，导入成功率最高。**
