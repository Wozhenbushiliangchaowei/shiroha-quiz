# Shiroha Quiz 原生 Compose 开发进度

## 项目概述

Shiroha Quiz 是一个自导入题库的刷题/考试 App，采用双入口架构：

| 入口         | 技术栈                                  | 包名                      | 状态   |
| ---------- | ------------------------------------ | ----------------------- | ---- |
| Web 壳子     | WebView 加载 HTML/CSS/JS               | `com.yiqiu.shirohaquiz` | 功能完整 |
| 原生 Compose | Kotlin + Jetpack Compose + Material3 | `com.reqir.shirohaquiz` | 开发中  |

## 原生页面（6 个）

| 页面   | 文件                    | 功能描述                                  |
| ---- | --------------------- | ------------------------------------- |
| 主页   | `HomeScreen.kt`       | 题库概览卡片、题库列表（切换/查看详情）、快速入口（导入/练习/考试）   |
| 导入页  | `ImportScreen.kt`     | 文件选择器、示例填入、标准文本解析、双文件（题目+答案）导入、识别结果预览 |
| 练习页  | `PracticeScreen.kt`   | 题卡展示（单选/多选/判断/填空/简答）、选项交互、答案提交与解析展示   |
| 考试页  | `ExamScreen.kt`       | 题量设置、计时器、自动交卷、得分统计、各题型正确率、明细报告        |
| 题库详情 | `BankDetailScreen.kt` | 题库内题目列表查看                             |
| 我的   | `MeScreen.kt`         | 个人设置/关于页面                             |

## 已完成模块

| 模块     | 文件数 | 说明                                                                                |
| ------ | --- | --------------------------------------------------------------------------------- |
| 导入解析器  | 8 个 | 标准题库解析、双文件合并、答案区解析、问卷星格式兼容、文本规范化                                                  |
| 评分与校验  | 2 个 | 导入策略评分、验证器                                                                        |
| 状态管理   | 1 个 | QuizRepository 单例：题库 CRUD、练习/考试状态、收藏/错题管理                                         |
| UI 组件库 | 1 个 | GlassCard、MetricGlassCard、ActionPillButton、QuizOptionCard、NoticeCard、StatusChip 等 |
| 主题系统   | 3 个 | Material3 主题、间距令牌(ShirohaSpacing)、字体                                              |
| 单元测试   | 2 个 | 解析器测试、考试流程测试                                                                      |

## 架构

```
MainActivity (入口路由)
  → ShirohaAppShell (底部导航栏 + 6 个页面)
    → HomeScreen / ImportScreen / PracticeScreen / ExamScreen / BankDetailScreen / MeScreen
```

- 所有页面共享 `QuizRepository` 单例状态
- 页面间通过回调导航（`onGoImport`、`onGoPractice` 等）
- 数据持久化：SharedPreferences（JSON 序列化）
