# C1驾照题库来源与说明

本版本默认题库已改为 **C1驾照科目一模拟练习题库**，不再使用任何行业内部题库作为示例或默认题库。

## 题库规模

- 总题数：393 题
- 单选题：127 题
- 判断题：254 题
- 多选题：12 题

## 题库定位

本题库用于通用刷题器的默认演示、导入测试和日常练习。题目围绕 C1 科目一常见考点整理，包括：

- 驾驶证与资格
- 车辆证件与车辆检查
- 速度规定
- 高速公路通行
- 灯光使用
- 让行规则
- 超车、变道、倒车
- 停车规定
- 事故处理
- 应急处置
- 安全文明驾驶
- 交通标志与标线

## 注意事项

1. 本题库不是公安交管部门官方最新全量题库。
2. 部分地区、年份、平台的题库会存在差异。
3. 正式备考时，建议以当地交管部门、驾校或权威驾考平台最新题库为准。
4. 本项目重点是测试通用刷题网站的题型、导入、错题本、考试模式等功能。
